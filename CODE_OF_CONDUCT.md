# Code of Conduct

## Our Pledge

We want Knob Swipe Navigation to be a welcoming project for Home Assistant users and contributors. Everyone who participates in issues, pull requests, discussions, reviews, and support conversations is expected to help keep the project respectful, useful, and safe.

## Expected Behavior

- Be kind, patient, and constructive.
- Assume good intent, especially when someone is reporting a confusing Home Assistant or device behavior.
- Keep feedback focused on the integration, the documentation, and the technical problem.
- Respect that logs, dashboard URLs, screenshots, and Home Assistant configuration can contain private information.
- Help people redact device ids, entity ids, URLs, and personal configuration before posting.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Sexualized language or imagery.
- Discriminatory comments or behavior.
- Publishing private information without explicit permission.
- Posting exploit details, private logs, dashboard URLs, or device identifiers in a way that could put another user at risk.

## Reporting

If you see behavior that violates this code of conduct, contact the maintainer privately through GitHub. Include the relevant links, screenshots, or context, and redact private Home Assistant data where possible.

Maintainers may remove comments, close discussions, block users, or take other moderation steps when needed to protect the project community.
